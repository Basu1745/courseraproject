# Code of Conduct

## Our Commitment
We are dedicated to providing a welcoming and harassment-free experience for everyone.

## Expected Behavior
✔ Be respectful and supportive  
✔ Use inclusive language  
✔ Accept constructive feedback  
✔ Help each other learn and grow  

## Unacceptable Behavior
✘ Harassment or discrimination  
✘ Offensive comments  
✘ Personal attacks or threats  

## Reporting Issues
If someone behaves inappropriately, please report it by opening an issue on the repository.

---

By participating in this project, you agree to follow this Code of Conduct.
